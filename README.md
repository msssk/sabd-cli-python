# Sabd CLI

Because:

    1. I need Gurbani on bash
    2. I need to learn Python

## Basic Usage
  `sabd-cli -f hjkk` to find "har jio kirpa karo" using first letter search

  `sabd-cli -s 2289` to bring all lines of "har jio kirpa karo" back

  `sabd-cli (-h | --help)`
